# Gastos

A simple webapp written in Python, Flask, MaterializeCSS and SQLite.

Steps:

1 - Create the virtual environment and install app dependencies
```bash
$ pip install -r requirements.txt
```

2 - Create the database
```bash
$ flask db upgrade
```

3 - Seed
```bash
$ flask seed
```

4 - Run the app
```bash
$ flask run
```
